### [77. 组合](https://leetcode-cn.com/problems/combinations/) (中等)

[le77.go](le77.go)

解题思路：c(n, k) = c(n-1, k) + c(n-1, k-1), 从 n-1里选k个（不含n），再从（n-1）里选（k-1）个（每个加上n）。
k>n||k<=0: 返回`[]`
k==n：返回`[1,2,....,n]`
k==1: 返回`[[1],[2]...[n]]`
