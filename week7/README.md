f(i, j, k, l)

### i, j

f(i, j)
决策：
* 买 f (i, 1) <- f(i-1, 0)-price[i]
* 卖 f (i, 0) <- f(i-1, 1)+price[i]
* 不动 f (i, 1) <- f(i-1, 0)

求max
f(0, 0) = -inf (-1e9)，（求min inf）
f (i) = max(f)

#### 空间优化

只有 i, i-1有用，使用滚动数组
第一步，每个i/i-1后边都加上&1

```go
f[i&1].. = f[(i-1)&1] ...
```
